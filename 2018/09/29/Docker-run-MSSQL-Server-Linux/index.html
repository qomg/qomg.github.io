Docker run MSSQL Server Linux

9月 29, 2018 8:44· 98 words · 1 minutes read

  1. pull
  2. run
  3. modify sa password
  4. restart (or start if your server has stop)
  5. check status
  6. as last

pull

docker pull microsoft/mssql-server-linux

run

docker run -e ‘ACCEPT_EULA=Y’ -e ‘SA_PASSWORD=‘ \
-p 1433:1433 –name docker-sql-server -d microsoft/mssql-server-linux

dont modify above string <YourStrong!PasswOrd>

modify sa password

docker exec -it docker-sql-server /opt/mssql-tools/bin/sqlcmd \
-S localhost -U SA -P ‘‘ \
-Q ‘ALTER LOGIN SA WITH PASSWORD=”YourNewStrong!PasswOrd”‘

you should replace YourNewStrong!PasswOrd with your password

restart (or start if your server has stop)

docker restart docker-sql-server

check status

docker ps -a

as last

connect server with sql operations studio or navicat
