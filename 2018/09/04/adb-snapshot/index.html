adb手机截图

1、
alias adbscreenshot=”adb -d shell screencap -p > $(date +”%Y-%m-%d%H-%M-%S”).png”
alias adbscreenshot=”adb -d shell screencap -p | perl -pe ‘s/\x0D\x0A/\x0A/g’ > $(date +”%Y-%m-%d%H-%M-%S”).png”

2、
adb exec-out screencap -p > screen.png

3、
adb shell screencap -p /sdcard/screencap.png
adb pull /sdcard/screencap.png

4、
adb shell screenrecord /sdcard/screen.mp4
^C 停止
adb pull /sdcard/screen.mp4

4.1、
adb shell screenrecord –bit-rate 12000000 /sdcard/screen.mp4
