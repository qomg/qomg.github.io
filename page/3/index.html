MacBook快捷键

按键 描述
空格键 预览
cmd + , 设置
cmd + -/= 缩小/放大
ctrl + u 删除到行首(与zsh冲突, zsh中是删除整行)
ctrl + k 删除到行尾
ctrl + p/n 上/下移动一行或者前/后一个命令
ctrl + b/f 光标前/后移char
esc + b/f 光标前/后移word
ctrl + a/e 到行首/行尾
ctrl + h/d 删前/后字符
ctrl + y 粘贴
ctrl + w 删除前一个单词
ctrl + d 删后一个单词
ctrl + _ 撤销
ctrl + r bck-i-search/reverse-i-search, 输入关键字搜索历史命令
